🔧 Enable Global Accounts (one-time setup)

To make accounts cross-device and the leaderboard truly global, connect a free Firebase project:

  1. Go to console.firebase.google.com → Create a new project
  2. In the project: Build → Firestore Database → Create database → Start in test mode
  3. Project Settings (⚙️) → Your apps → Add a Web app → copy the firebaseConfig object
  4. In this HTML file, replace the FIREBASE_CONFIG block near the top of the <script> with your config
  5. For Google Sign-In: Authentication → Sign-in method → Google → Enable

That's it — free tier supports thousands of users. Once configured, all logins, stats and the leaderboard are global.
